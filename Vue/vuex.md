# vuex

# 什么是vuex

专门在Vue中实现集中式状态(数据)管理的一个Vue插件, 对Vue应用中多个组件的共享状态进行集中式的管路 (读写), 也是一种组件间通信的方式, 且适用于任意组件的通信

# 什么时候使用vuex

* 多个组件依赖于同一状态
* 来自不同组建的行为需要变更同一状态
