# 插件

功能: 用于增强Vue

本质: 包含`install`​方法的一个对象, `install`​的第一个参数是`Vue`​, 第二个以后的参数是使用插件时传入的参数

# 使用

定义插件: 

新建`plugin.js`

```js
import { mixin } from "../components/mixin";

export const plugin = {
    install(Vue) {
        //添加全局过滤器
        Vue.filters();

        //添加全局指令
        Vue.directive();

        // 添加全局混入
        Vue.mixin(mixin);

        // 添加实例方法
        Vue.prototype.myMethod = () => {
            console.log("method");
        };
    },
};

```

使用插件: 

```js
Vue.use(plugin);
```
