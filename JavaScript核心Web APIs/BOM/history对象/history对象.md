# history对象方法

| history对象方法 | 含义                             |
| ----------- | ------------------------------ |
| back()      | 后退                             |
| forward()   | 前进                             |
| go(n)       | 如果n为正数,表示前进n个页面, 如果为负,表示后退n个页面 |
