# 页面跳转传递数据

效果: 第一个页面输入用户名, 第二个网页显示

思路: 第一个页面使用form action直接提交, 不需要js代码

第二个页面首先通过location.search属性获取参数, 然后通过substr去掉前面的?&#x20;

最后通过split分割字符串来取到信息

代码:

login.html

```javascript
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>

        <style>
            body {
                background-color: #ccc;
            }
            .wrap {
                width: 200px;
                margin: 200px auto;
            }
        </style>
    </head>
    <body>
        <div class="wrap">
            <form action="index.html">
                <input class="text" type="text" name="username" />
                <input class="btn" type="submit" value="提交" />
            </form>
        </div>
    </body>
</html>

```

index.html

```javascript
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body>
        <h1>欢迎:</h1>
        <div class="username">tim</div>
        <script>
            var div = document.querySelector(".username");
            var username = window.location.search;

            username = username.substr(1); //去掉问号

            username = username.split("="); //分割字符串
            console.log(username);

            div.innerHTML = username["1"];
        </script>
    </body>
</html>

```
