# jQuery 属性

# 设置或获取元素固有属性值 prop()

所谓元素固有属性就是元素本身自带的属性, 例如`<a>`标签的`href`, `<input>`里面的`type`

## 获取属性

`prop("属性");`

## 设置属性

`prop("属性",属性值)`

‍

# 设置或获取元素自定义属性值 attr()

用户给元素添加的属性, 我们称之为自定义属性, 比如给`<div>`添加`index="1";`

## 获取属性语法

`attr("属性"); ​`

## 设置属性语法

`attr("属性","属性值");`

# 数据缓存 data()

data()方法可以在指定的元素上存取数据, 并不会修改DOM元素结构, 一旦页面刷新, 之前存放的数据都将被移除

## 设置数据

`data("属性","属性值");`

## 获取数据

`data("属性");`

此外. 还可以读取HTML5自定义属性 data-index, 得到的是数字型

‍
