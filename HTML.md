# HTML

HTML(Hyper Text Markup Language) 超文本标记语言

# HTML 结构:

```html
<html>
  <head>
    <title>标题</title>
  </head>
  <body>主体</body>
</html>
```

‍

# HTML 标签结构

1. <开始标签>包裹内容</结束标签>
2. 常见的标签由两部分组成,开始标签和结束标签.
