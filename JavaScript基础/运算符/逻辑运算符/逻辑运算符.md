# 逻辑短路 ( 逻辑中断 )

当有多个表达式时, 如果左面的表达式可以确定逻辑表达式的值时, 就不再计算右边的值

## 逻辑与短路运算

表达式1 && 表达式2

*   如果表达式1为真, 那么返回表达式2的值

*   如果表达式1为假, 那么返回表达式1的值

```javascript
        console.log(123&&456); //456
        console.log(0&&456); //0
```

## 逻辑或短路运算

表达式1 || 表达式2

*   如果表达式1为真, 那么返回表达式1的值

*   如果表达式1为假, 那么返回表达式2的值

```javascript
        console.log(0||456); //456
        console.log(1||456); //1
```
