# while循环

while语句可以在条件表达式为真的前提下, 循环执行指定的一段代码, 知道表达式不为真时结束循环

# 语法结构

```JavaScript
while(表达式){
//循环体
}
```

执行流程:

1. 先判断条件表达式, 如果结果为真, 执行循环体, 如果为假, 退出循环
2. 执行循环体
3. 回到1

注意:

* 循环体里面应该有计数器更新代码, 否则会产生死循环
