# do...while循环

do...while语句其实是while语句的一个变体, 该循环会先执行一次代码块, 然后对表达式进行判断, 如果条件为真, 就会重复执行循环体, 否则退出循环

```JavaScript
            i = 0;
            do {
                console.log(i++);
            } while (i < 100);
```

执行过程:

1. 先执行一遍循环体
2. 进行判断,如果为真, 继续循环, 如果为假, 推出循环

do...while至少会执行一次循环体
