# 输入输出语句

# 输入

## prompt(info)

* 浏览器弹出输入框, 用户可以输入
* 第一个参数: 用户提示语
* 第二个参数: 输入框的默认值

## console.dir(对象)

打印元素对象, 查看里面的属性和方法

```JavaScript
    <div id="time">2022/5/1 14:44</div>
    <script>
        var time = document.getElementById("time");
        console.dir(time);
    </script>
```
