# 节流阀

目的:  防止动画函数执行速度过快的问题

思路: 当上一个动画函数执行完毕, 再去执行下一个动画函数, 让事件无法持续触发

方法:  利用回调函数, 添加一个变量tag, 锁住和解锁函数

在函数外部定义一个tag=true;

在函数内部开始执行时先判断tag是否为真, 如果为真才能执行,

设置tag=false, 在动画函数的回调函数内设置tag=true, 这样就实现了只有动画函数执行完毕才能执行下一个动画函数

```JavaScript
var tag = true;
function fun (){
if(tag){
  tag=false;
  animate(obj,100,callback())
  }
}
callback=function(){
  tag=true;
}
```
