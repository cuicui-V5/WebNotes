# 元素跟随鼠标

效果: 某个元素跟随鼠标进行移动

思路: 利用mousemove获取鼠标移动事件, 利用事件对象获取鼠标的坐标,  给图片设置绝对定位, 然后更改他的top和left值即可

代码:

```JavaScript
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
        <style>
            .cursor {
                width: 50px;
                height: 50px;
                position: fixed;
                top: 100px;
                left: 100px;
            }
            .cursor img {
                width: 100%;
                height: 100%;
            }
        </style>
    </head>
    <body>
        <div class="cursor">
            <img src="./image/funny.jpg" alt="" />
        </div>
        <script>
            var cu = document.querySelector(".cursor");
            console.log(cu);
            cu.style.left = "200" + "px";
            cu.style.top = "200" + "px";
            document.addEventListener("mousemove", function (e) {
                x = e.clientX;
                y = e.clientY;
                console.log(x);
                cu.style.left = x + "px";
                cu.style.top = y + "px";
            });
        </script>
    </body>
</html>

```
